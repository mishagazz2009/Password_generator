## Описание программы

Random Password Generator — это GUI-приложение на Python для генерации безопасных случайных паролей. Программа позволяет создавать пароли с настраиваемыми параметрами, сохранять историю сгенерированных паролей и управлять ею. Интерфейс построен на библиотеке Tkinter, все данные хранятся в формате JSON.

Приложение будет полезно:
- Для создания надежных паролей к аккаунтам
- Для генерации временных паролей
- Для тестирования безопасности
- В образовательных целях

---

## Возможности программы

### Основные функции:
- **Генерация паролей** с использованием модуля `random`
- **Настройка параметров пароля:**
  - Длина от 4 до 32 символов (ползунок)
  - Заглавные буквы (A-Z)
  - Строчные буквы (a-z)
  - Цифры (0-9)
  - Специальные символы (!@#$%^&*()_+-=[]{}|;:,.<>?)
- **Гарантированное наличие** выбранных типов символов в пароле
- **Копирование в буфер обмена** сгенерированного пароля
- **Сохранение истории** паролей в JSON-файл
- **Таблица истории** с датой создания, паролем и длиной
- **Управление историей:**
  - Копирование паролей из истории
  - Удаление отдельных записей
  - Полная очистка истории
- **Автозагрузка истории** при запуске программы
- **Автосохранение** при добавлении/удалении записей

## Проверка корректности ввода

Программа включает следующие проверки:

### 1. Проверка выбора типов символов
### 2. Проверка минимальной/максимальной длины
### 3. Проверка перед копированием
### 4. Проверка перед сохранением в историю
### 5. Проверка перед удалением
### 6. Проверка загрузки истории
### 7. Проверка выбора в таблице истории
